# 🏋️‍♂️ Desafio Academia - Controle de Carga (Java)

Este é um programa simples em Java que simula um assistente de treino interativo e motivacional. O projeto foi desenvolvido com o objetivo de praticar os conceitos fundamentais da linguagem e treinar a lógica de programação.

## 🚀 Conceitos Aplicados no Código

O programa utiliza recursos essenciais para quem está evoluindo no Java:
* 📥 **Scanner (`teclado`):** Captura o nome do aluno e a carga do exercício direto pelo terminal.
* 🔀 **Estrutura Condicional (`if / else`):** Avalia o peso que o usuário vai levantar para definir a resposta do assistente.
* 🔁 **Estrutura de Repetição (`for`):** Cria um loop que conta de 1 até 5 para simular as repetições do exercício.
* 🎯 **Java Moderno:** Desenvolvido utilizando *Unnamed Classes* (método `main` direto, sem a necessidade de declarar uma classe formal em volta), deixando o código limpo e focado no que importa.

## 🖥️ Como o Programa Funciona

1. **Entrada de Dados:** O programa pede o seu nome e a carga (em kg) que você vai levantar.
2. **O Desafio (`if`):** Se você digitar uma carga **maior ou igual a 100 kg**, o assistente diz que quer ver do que você é capaz e o loop `for` entra em ação, contando na tela as repetições de **1 a 5**, terminando com um "nada mal".
3. **A Provocação (`else`):** Se a carga for **menor que 100 kg**, o programa cai na condição alternativa e exibe a mensagem: *"so isso? é pra aquecer ne?"*.

## 🛠️ O Aprendizado nos Bastidores

Embora o código final seja direto e limpo, o verdadeiro valor deste projeto foi o processo de desenvolvimento. Retornar ao Java exige atenção a detalhes que outras linguagens escondem, como:
* O uso correto dos pontos e vírgulas `;` no final das instruções.
* O fechamento correto das chaves `{}` para abrir e fechar os blocos do `if`, `else` e `for`.
* A sintaxe específica do loop `for(int contador = 1; contador <=5; contador++)`.

Superar esses pequenos erros de sintaxe linha por linha é o que constrói a base de um bom programador!
